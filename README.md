# NeptuneApex

A Ruby Gem for interacting with a Neptune Apex aquarium controller.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'neptune_apex'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install neptune_apex
